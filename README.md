# 🐍 Ejercicios de Fundamentos de Python

## 📋 Descripción
Colección de ejercicios prácticos para aprender y reforzar los fundamentos de Python.

## 📂 Estructura del Proyecto
```
fundamentos-python/
│
├── 01_variables/
├── 02_entrada_de_datos/
├── 03_metodos_str/
├── 04_operadores/
├── 05_booleanos/
├── 06_sentencias/
├── 07_bucle_while/
├── 08_Listas/
├── 09_Metodos_listas/
├── 10_Objeto_range/
├── 11_Tuplas/
├── 12_Diccionarios/
└── 13_Funciones/
```

## 🚀 Requisitos
- Python 3.8 o superior

## ▶️ Cómo ejecutar los ejercicios
1. Clona el repositorio
2. Navega a la carpeta del ejercicio deseado
3. Ejecuta el archivo con:
   ```bash
   python nombre_ejercicio.py
   ```

## 📚 Temas cubiertos
- Variables y tipos de datos
- Operadores aritméticos, lógicos y de comparación
- Estructuras condicionales (`if`, `elif`, `else`)
- Bucles (`for`, `while`)
- Funciones y parámetros
- Listas, tuplas y diccionarios
- Entrada y Salida de datos
